# 📑 Installation

## 1. Clone repository
```bash
git clone https://github.com/your-username/McLium.git
cd McLium
```

## 2. Create virtual environment (recommended)

### Linux / macOs
```bash
python -m venv .venv
source .venv/bin/activate   
```

### Window
```bash
python -m venv .venv
.venv\Scripts\activate 
```

## 3. Run McLium
```bash
python Mclium.py --help
```
