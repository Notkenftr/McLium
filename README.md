# 📑 Installation

## 1. Clone repository
```bash
git clone https://github.com/your-username/McLium.git
cd McLium
```

## 2. Install
### Linux / macOs
```bash
chmod +x ./install.sh
./install.sh
```

### Window
```bash
install.bat
```

## 3. Run McLium
```bash
mclium --help
```
